#### Prerequisites
```
dotnet --version
2.1.403
```
```
npm --version
5.6.0
```
#### Run
```
git clone https://github.com/edgythorn/bookstore.git
cd bookstore/BooksStore.Host/ClientApp
npm install
cd ..
dotnet run
```
When the application will be launched open url http://localhost:5000 in your browser.
